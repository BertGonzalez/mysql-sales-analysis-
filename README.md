Proyecto SQL: Análisis de ventas en MySQL

Descripción

Este proyecto reúne una serie de consultas SQL desarrolladas sobre una base de datos de ventas con el objetivo de practicar y fortalecer habilidades de análisis de datos en MySQL.
A través de distintas consultas, se exploran ventas, envíos, productos y vendedores, respondiendo preguntas de negocio mediante filtros, columnas calculadas, lógica condicional, uniones entre tablas y funciones de agregación.

Objetivos del proyecto:

- Consultar y explorar una base de datos de ventas
- Practicar filtros y ordenamiento de información
- Crear columnas calculadas y alias
- Aplicar lógica condicional con CASE
- Relacionar tablas mediante JOIN
- Responder preguntas de negocio usando SQL
- Utilizar funciones de agregación como COUNT() y SUM() 

Herramientas utilizadas

- MySQL 
- SQL
   
Conceptos aplicados

En este proyecto se trabajó con:

- SELECT
- WHERE 
- ORDER BY 
- columnas calculadas 
- alias con AS 
- filtros por fecha con YEAR(), BETWEEN() y WEEKDAY() 
- lógica condicional con CASE 
- JOIN 
- GROUP BY 
- funciones de agregación como COUNT() y SUM()

Preguntas de negocio resueltas

Algunos ejemplos de análisis realizados:

- ¿Qué ventas fueron superiores a cierto monto?
- ¿Cuántos envíos se realizaron en enero de 2022?
- ¿Qué producto vendió más cajas: Milk Bars o Eclairs?
- ¿Qué ventas correspondieron a ciertos rangos de cajas, clientes o fechas?
- ¿Qué envíos cumplieron determinadas condiciones comerciales? 

Tablas utilizadas

- sales
- people 
- products
  
Resultado

Este proyecto me permitió seguir fortaleciendo mis fundamentos en SQL y practicar el uso de consultas para transformar datos en información útil para el análisis de negocio.
